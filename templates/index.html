🎓 Classroom Wordle

A word guessing game for the classroom!

👨‍🏫 Teacher Interface 🎒 Student Game 🔍 Student Search 🏆 Leaderboard

How to Play:

  • 🎯 Teacher sets a 5-letter word
  • 📝 Students have 6 attempts to guess it
  • 🟩 Green = correct letter in right position
  • 🟨 Yellow = correct letter in wrong position
  • ⬜ Gray = letter not in word
